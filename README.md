# ShaoShuai0605.github.io

GitHub Pages academic homepage for Shuai Shao.

## Files

- `index.html`: page structure and content
- `styles.css`: visual system and layout
- `assets/CV_ShuaiShao.pdf`: downloadable CV used by the site
- `DESIGN.md`: local design brief derived from an awesome-design-md Mintlify-inspired style

## Publish

This repository is set up for a user site. Once GitHub Pages is enabled on the `main` branch root,
the homepage will publish at:

`https://shaoshuai0605.github.io`
